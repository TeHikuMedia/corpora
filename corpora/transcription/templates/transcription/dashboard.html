Kia ora!

Dashboard

Here you'll find links to a browseable, interactive API that let's you test the API using HTML form inputs. The Docs provide examples on how to use the API in python or on the command line. We have a Speak Demo that let's you test the te reo Māori speech recognition on your voice. And finally, we've built a Transcription Tool using the API to help shorten the time it takes to transcribe te reo Māori audio recordings.

Transcriptions

You can manage of all the audio you upload to {{site.domain}} in the Transcriptions panel. You can also edit transcriptions, which are broken into caption-friendly segments for audio that's longer than 10 seconds. We'll add some download features soon so you can download your transcriptions into useable formats such as webvtt.

Speak Demo

Because hey, we all want to have a tutu. This is just a rough demo to let you talk and see what the computer thinks you said. The demo doesn't work on all browsers or devices. We'll leave it to you amazing developers to go build some native apps.

API

We're releasing a very basic API so that developers can get started with te reo Māori speech recognition. If you have any feature requests or come accross bugs, please file them on our GitHub page. {{site.domain}} is a part of our Kōrero Māori project, and we've "open sourced" this project under our Kaitiakitanga License. You can read a very early draft of that license here.
