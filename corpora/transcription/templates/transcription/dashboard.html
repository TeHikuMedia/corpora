API

We're releasing a very basic API so that developers can get started with te reo Māori speech recognition. If you have any feature requests or come accross bugs, please file them on our GitHub page. {{site.domain}} is a part of our Kōrero Māori project, and we've "open sourced" this project under our Kaitiakitanga License. Basic Usage

Below is a simple example of posting audio to the API transcription endpoint.

curl -X POST -k https://{{site.domain}}/api/transcription/ -H "Authorization: Token YOUR-UNIQUE-TOKEN" -F audio_file=@audio.mp3

{
  "uploaded_by": 1,
  "audio_file": "https://...3e06.mp3",
  "pk": 279,
  "name": null,
  "transcription": ""
}

The response will include a link to the file you posted as well as a private key (pk). Use the pk when making a GET request to the end point so you can get the transcription. Depending how long your audio is, it might take some time to transcribe. If you want to get the transcription back in the same POST request, use the ?method=stream key word argument. Currently we only support audio of 10 seconds or less with ?method=stream.

Transcriptions

You can manage of all the audio you upload to {{site.domain}} in the Transcriptions panel. You can also edit transcriptions, which are broken into caption-friendly segments for audio that's longer than 10 seconds. We'll add some download features soon so you can download your transcriptions into useable formats such as webvtt.
